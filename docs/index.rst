Welcome to django-rateslide's documentation!
============================================

`rateslide <https://github.com/JanOosting/django-rateslide>`_ is a Django app that organizes case lists of cases with microscope slides.

the `django-histoslide <https://github.com/JanOosting/django-histoslide>`_ app is used to handle the slides.

Case lists can have multiple purposes like showing cases, rating cases with structured questionaires, or examinations.
